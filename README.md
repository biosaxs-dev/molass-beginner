# molass-beginner

An Agent-mode onboarding repository for new molass users.

Open this folder in VS Code with GitHub Copilot, then simply say:

> **"I want to begin with molass"**

The agent will guide you from there — no prior knowledge required.

---

## What to say next

After the initial setup, here are example phrases to continue:

**Load data**
> "Run the analysis on SAMPLE1"  
> "I have my own data in `/path/to/my/data` — can you load it?"  
> "What other sample datasets are available in molass_data?"

**Explore your results**
> "What does the Rg value tell me about my protein?"  
> "Why does the 3D plot look like a mountain range?"  
> "What do the decomposed components represent?"

**Understand what happened**
> "Explain what baseline correction does"  
> "Why do we trim the data before analysis?"  
> "What is SEC-SAXS and why is decomposition needed?"

**Go deeper**
> "Show me the full tutorial"  
> "How do I get Rg as a function of elution frame?"  
> "How do I export the results?"
